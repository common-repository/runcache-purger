== Description ==

RunCache Purger allow you to purge RunCloud NGINX FastCGI, Proxy Cache and Redis Object Cache for better performance. 

This plugin need to be installed directly from RunCloud Panel. It will not work if you download and install it manually via WordPress Panel.

This plugin has been deprecated. Please install our latest RunCloud Hub plugin through RunCloud Panel to enjoy server side caching, cache exclusion and better performance.

== Installation ==

This plugin need to be installed directly from RunCloud Panel. It will not work if you download and install it manually via WordPress Panel.
